# Desafio 00: Introdução ao GitHub Copilot (Somente Leitura)

### Duração Estimada: 20 Minutos

>**Observação:** Este desafio é apenas para leitura e exploração.

O GitHub Copilot é a primeira ferramenta de desenvolvimento com inteligência artificial em larga escala do mundo, criada para ajudar você a escrever código mais rápido e com menos esforço. Ele utiliza comentários e o contexto do código para sugerir, de forma instantânea, linhas completas e até mesmo funções inteiras.
Pesquisas mostram que o GitHub Copilot ajuda desenvolvedores a programar com mais agilidade, manter o foco em problemas complexos, preservar o fluxo de trabalho por mais tempo e a se sentirem mais satisfeitos com o resultado do próprio trabalho.

Baseado no OpenAI Codex — modelo de linguagem generativo e pré-treinado desenvolvido pela OpenAI — o GitHub Copilot está disponível como extensão para o Visual Studio Code, Visual Studio, Neovim e para a suíte de IDEs da JetBrains.

## Github Copilot for Individuals

O GitHub Copilot é um assistente de programação baseado em inteligência artificial, desenvolvido pelo GitHub, projetado para auxiliar desenvolvedores individuais por meio de sugestões de código inteligentes, automação de tarefas repetitivas e aumento da produtividade no desenvolvimento de software.

Ele utiliza modelos de aprendizado de máquina, treinados em grandes repositórios de código, para gerar trechos de código, completar funções e fornecer sugestões contextuais com base na entrada do desenvolvedor e nos seus padrões de codificação. O Copilot suporta diversas linguagens de programação e se adapta ao estilo de cada usuário, tornando-se uma ferramenta valiosa para acelerar tarefas de programação, explorar novas técnicas e melhorar a qualidade do código.

Formas como os dados no GitHub Copilot Individual podem ser usados e compartilhados:

- **Aprimorando o GitHub Copilot**: Os dados coletados são utilizados para melhorar o GitHub Copilot, avaliando diferentes estratégias para processar e prever sugestões que os usuários possam considerar valiosas.
- **Desenvolvimento de produtos e serviços relacionados para desenvolvedores**: Os insights obtidos a partir dos dados ajudam no desenvolvimento e na melhoria de outras ferramentas e serviços para desenvolvedores oferecidos pelo GitHub e pela Microsoft.
- **Detecção de abusos e violações de políticas**: Os dados são examinados para investigar e identificar qualquer possível uso indevido ou violação das Políticas de Uso Aceitável associadas ao GitHub Copilot.

## GitHub Copilot for Business

O GitHub Copilot está disponível por meio de contas pessoais do GitHub, com o GitHub Copilot for Individuals, ou através de contas organizacionais ou empresariais, com o GitHub Copilot for Business.

Com o Copilot for Business, você pode gerenciar o acesso ao GitHub Copilot para as organizações dentro da sua empresa. Uma vez que você concede a uma organização acesso ao GitHub Copilot, os administradores da organização podem conceder acesso a indivíduos e equipes.

Com o GitHub Copilot for Business, o GitHub Copilot está aberto a todos os desenvolvedores, equipes, organizações e empresas.

Focado em tornar as organizações mais produtivas, seguras e satisfeitas, o GitHub Copilot for Business permite que os desenvolvedores programem mais rápido e se concentrem em trabalhos mais gratificantes.

Aqui estão alguns recursos que você pode esperar do Copilot for Business:

 - **Um modelo de IA robusto**: Novos algoritmos de modelagem melhoram a qualidade das sugestões de código.
 - **Filtragem de vulnerabilidades de segurança baseada em IA**: O GitHub Copilot bloqueia automaticamente sugestões de código inseguras comuns, visando problemas como credenciais fixas no código (hardcoded), injeções de SQL e injeções de caminho.
 - **Suporte a proxy VPN**: O GitHub Copilot funciona com VPNs, incluindo certificados autoassinados, para que os desenvolvedores possam usá-lo em qualquer ambiente de trabalho.
 - **Inscrição simplificada**: Independentemente de uma empresa usar ou não o serviço do GitHub para seu código-fonte, ela pode facilmente adquirir licenças do Copilot for Business online e atribuir os acessos.

## GitHub Copilot for Enterprise

O GitHub Copilot Enterprise é um plano do Copilot disponível para empresas que usam o GitHub Enterprise Cloud. Ele fornece recursos de IA para aprimorar sua experiência no GitHub.com, como a capacidade de conversar com o Copilot no navegador e referenciar o contexto de todos os repositórios do seu projeto.

Os proprietários da empresa podem permitir que algumas ou todas as organizações na empresa acessem o GitHub Copilot. Se uma organização tiver acesso ao Copilot, os proprietários da organização podem atribuir vagas do Copilot Enterprise a alguns ou a todos os membros da organização.

### O GitHub Copilot Enterprise inclui os seguintes recursos, disponíveis para todos os usuários que recebem uma vaga do GitHub Copilot Enterprise:

- **GitHub Copilot no seu IDE**: O GitHub Copilot é uma ferramenta de IA que oferece sugestões no estilo de autocompletar no seu IDE.
- **GitHub Copilot Chat no seu IDE**: O GitHub Copilot Chat no seu IDE é uma interface de chat que permite fazer perguntas e receber respostas relacionadas à programação dentro dos IDEs suportados.
- **GitHub Copilot no CLI**: O GitHub Copilot no CLI fornece uma interface de chat no terminal, permitindo que você faça perguntas sobre a linha de comando. Você pode pedir ao GitHub Copilot para fornecer sugestões de comandos ou explicações sobre comandos específicos.
- **GitHub Copilot Chat no GitHub.com**:  GitHub Copilot Chat no GitHub.com é uma interface de chat que permite fazer perguntas e receber respostas relacionadas a programação dentro do GitHub.com.
- **Resumos de pull requests do Copilot**: Os resumos de pull requests do Copilot ão um recurso baseado em IA que permite criar um resumo das alterações feitas em um pull request, quais arquivos elas impactam e no que um revisor deve focar ao conduzir sua revisão.

### Casos de uso para GitHub Copilot for Business 

**GitHub Copilot** melhora a produtividade e a satisfação dos desenvolvedores, reduz interrupções, melhora o fluxo de trabalho e aumenta o tempo que um desenvolvedor passa realizando um trabalho gratificante. A seguir, estão os casos de uso de como o **GitHub Copilot for Business** pode ser utilizado:

1. **Automatização de Tarefas Rotineiras:** Um desenvolvedor em uma empresa de software frequentemente se vê escrevendo trechos de código semelhantes para diferentes projetos. Com o GitHub Copilot, o desenvolvedor pode automatizar esse processo, economizando tempo e aumentando a produtividade.

1. **Integração de Novos Membros da Equipe:** Uma empresa de tecnologia contrata um novo desenvolvedor que não está familiarizado com a base de código (codebase) da empresa. O GitHub Copilot pode ajudar o novo contratado a entender a base de código mais rapidamente, fornecendo sugestões e melhorando sua eficiência.

1. **Melhora a Qualidade do Código:** Uma empresa deseja melhorar a qualidade de seu código e reduzir erros. Com o GitHub Copilot, os desenvolvedores podem obter sugestões de melhores práticas e ter um "segundo par de olhos" para identificar possíveis problemas.

1. **Aceleração da Entrega de Projetos:** Uma empresa de serviços de TI está trabalhando com um prazo apertado para um projeto de cliente. Usando o GitHub Copilot, a equipe pode acelerar o processo de desenvolvimento, garantindo que cumpram o prazo sem comprometer a qualidade do código.
   
1. **Aprimorando a Codificação Colaborativa:** Uma empresa global com desenvolvedores trabalhando em diferentes locais geográficos usa o GitHub Copilot para garantir a consistência nas práticas de codificação em toda a equipe. Isso melhora a colaboração e reduz a probabilidade de conflitos ou mal-entendidos.

1. **Construção Rápida de um Protótipo:** Uma startup deseja demonstrar um protótipo de seu aplicativo para potenciais investidores. O GitHub Copilot pode ajudá-los a gerar rapidamente o código necessário, permitindo que se concentrem na lógica de negócios e nos aspectos exclusivos de seu aplicativo.

### Hablitar o GitHub Copilot Enterprise

Para permitir que os usuários utilizem os recursos do Copilot Enterprise, sua GitHub Enterprise Cloud enterprise deve ter uma assinatura do Copilot Enterprise.

O acesso ao Copilot é determinado nos níveis de empresa e organização:

- Os usuários com a função de `Enterprise Owner` podem conceder acesso ao GitHub Copilot a algumas ou a todas as organizações dentro da empresa.
- Os usuários com a função de `Enterprise Owner` que têm acesso ao Copilot podem atribuir licenças do GitHub Copilot for Enterprise a alguns ou a todos os membros de sua organização. 

Quando o acesso ao Copilot é habilitado, recursos como a capacidade de usar o Copilot no GitHub.com podem ser configurados. Os recursos são habilitados ou desabilitados no nível da empresa ou da organização:

- Usuários com a função de `Enterprise Owner` podem definir quais recursos do Copilot serão habilitados ou desabilitados para todas as organizações da empresa que têm acesso ao Copilot. Alternativamente, eles podem permitir que cada organização defina suas próprias políticas para cada recurso.
- Se os usuários com a função de `Enterprise Owner` permitirem, os proprietários das organizações poderão ativar ou desativar os recursos do Copilot para suas respectivas organizações.
